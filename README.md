# puzzle
puzzle solving tool

This repository contains early version of puzzle-solving tool.

For now, only puzzles of numbers are supported. As example:

|   |   |   |   |   |   |   |   |   |
|---|---|---|---|---|---|---|---|---|
|5| 4| 6| 2| 1| 3| 8| 7| 9| 

Is solved to:

|   |   |  |
|---|---|---|
| 1 | 2 | 3 |
| 4 | 5 | 6 |
| 7 | 8 | 9 |
